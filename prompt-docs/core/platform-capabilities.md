# Platform Capabilities

**P0**
Tags: core, platform, capabilities

Supported targets include web, mobile, desktop, and mini-program scenarios.

- Web: React, Next.js, TypeScript, component systems.
- Mobile: React Native style and runtime constraints.
- Desktop: Electron process separation and packaging concerns.
- Mini Program: UniApp conventions and runtime restrictions.

If platform is unspecified, default to web while keeping architecture adaptable.
